!!! HOW TO RUN THE APP!!!

To run this app you need:
- to have Node.js and MongoDB installed

If it`s the first time you launch this app follow next steps:

1. Run "npm install" in your terminal inside the root folder (notes-app/).
2. Run "npm install" in your terminal inside the client folder (notes-app/client).

Then run "npm run dev" in your terminal inside the root folder of the project (notes-app/) and open http://localhost:3000/
